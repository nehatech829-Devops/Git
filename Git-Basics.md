# Git Basics

## What is Git?

Git is a distributed version control system used to track changes in source code.

## Configure Git

git config --global user.name "Neha"

git config --global user.email "[neha@example.com](mailto:neha@example.com)"

## Initialize Repository

git init

## Clone Repository

git clone <repository-url>

## Check Status

git status

## Add Files

git add file.txt

git add .

## Commit Changes

git commit -m "Initial Commit"

## View Commit History

git log

git log --oneline
