# Git Branching

## Create Branch

git branch feature

## List Branches

git branch

## Switch Branch

git checkout feature

OR

git switch feature

## Create and Switch

git checkout -b feature

## Delete Branch

git branch -d feature

## Rename Branch

git branch -m old-name new-name

## Why Branching?

* Parallel development
* Feature isolation
* Safe experimentation
* Easier collaboration
